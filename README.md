# VaxNear

A location-based vaccination finder, record keeper, and travel vaccine planner for iOS.

## Features

- **Find** — Discover nearby pharmacies, clinics, and health departments offering vaccinations
- **Records** — Keep a complete timeline of your immunization history, synced with HealthKit
- **Travel** — Plan ahead with destination-specific vaccine recommendations powered by CDC data
- **Settings** — Face ID lock, iCloud sync, HealthKit integration

## Requirements

- iOS 17.0+
- Xcode 16+

## Getting Started

1. Open `VaxNear.xcodeproj` in Xcode
2. Select your development team
3. Build and run on a simulator or device
